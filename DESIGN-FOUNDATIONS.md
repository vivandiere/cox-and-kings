#### Accent — Soft Cream Deep (Design Default)

The accent color with tint variations generated by mixing toward stone and primary. These values are for `Soft Cream Deep #9A8A5A`. In Figma, create these as a variable collection so the accent can be swapped.

| Token | Hex | Mix | Usage |
|---|---|---|---|
| `accent/120` | Mixed 35% → primary | Darkened | High-contrast text, active labels, tag text |
| `accent/100` | `#9A8A5A` | Base | Full accent — icons, labels, CTAs |
| `accent/80` | Mixed 15% → stone | 85% accent | Headings on light bg, search prompt text |
| `accent/60` | Mixed 30% → stone | 70% accent | Secondary accent elements |
| `accent/50` | Mixed 45% → stone | 55% accent | Medium — decorative borders |
| `accent/40` | Mixed 55% → stone | 45% accent | Subtle highlights |
| `accent/20` | Mixed 72% → stone | 28% accent | Tag backgrounds, pill fills |
| `accent/10` | Mixed 82% → stone | 18% accent | Dividers, faint backgrounds |

> **Mixing formula:** `result = base + (target - base) × amount` per RGB channel.
> Mix toward stone `#F2F2EB` for lighter tints, toward primary `#102037` for darker shades.

## 3. Effects & Animation

Reference for Figma prototyping and developer handoff.

| Effect | Properties | Usage |
|---|---|---|
| **Card shadow** | `0 1px 3px rgba(16,32,55, 0.08)` | Cards, elevated surfaces |
| **Nav shadow** | `0 2px 12px rgba(16,32,55, 0.12)` | Compact floating nav |
| **Hover darken** | Primary overlay at 25% opacity | Destination shapes, cards |
| **Image zoom** | `scale(1.05)`, 500ms ease | Region shape images on hover |
| **Fade in** | `opacity 0→1`, 600ms ease | Images appearing in active slider position |
| **Slide transition** | `transform translateX`, 800ms `cubic-bezier(0.4, 0, 0.2, 1)` | Destinations shape slider |
| **Stagger entrance** | `opacity 0→1, y 12→0`, 400ms, 60ms stagger | Menu items on open (GSAP) |
| **Blinking cursor** | `opacity 1→0→1`, 1s ease-in-out infinite | Search input idle state |
| **GPS ping** | Two rings scaling `1→2.4×` and `1→3.2×`, fading out, 2.4s loop, 600ms offset | "You are here" indicator |
| **Hamburger → X** | Bars rotate ±45° and translate, 300ms ease | Menu open/close toggle |
| **Hero auto-advance** | 5s interval, crossfade images | Hero image carousel |
| **Destinations auto-advance** | 4s interval, translateX shift | Region shapes slider |

## 6. Quick Reference — Key Decisions

| Decision | Value | Rationale |
|---|---|---|
| Primary color | Navy `#102037` | Heritage brand, authority, travel luxury |
| Surface color | Warm stone `#F2F2EB` | Avoids clinical white, pairs with navy |
| Accent (design default) | Soft Cream Deep `#9A8A5A` | Legible ochre — warm, luxe, good contrast |
| Type ratio | Perfect Fourth (1.333) | Classic, not too dramatic, works for editorial |
| Heading font | MesoTRIAL | Serif with character, editorial luxury |
| Body font | CenoTRIAL | Clean sans-serif, excellent readability |
| Border radius | 0px (buttons), 12px (cards) | Sharp buttons = premium, rounded cards = approachable |
| Spacing base | 8px grid | Industry standard, flexible |
| Nav transition | 40px scroll threshold | Quick enough to feel responsive |
